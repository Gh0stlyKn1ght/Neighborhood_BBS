## рҹ“Ӣ Overview

**Neighborhood BBS** is a fully open-source, decentralized community board and real-time chatroom platform designed to run on affordable, low-power hardware like the **Raspberry Pi**, **Zima Board**, and **ESP8266**. Create a local neighborhood network where residents can share information, ask questions, organize events, and build community connectionsвҖ”all while keeping data local and secure.

```ini
host = 0.0.0.0
port = 8080

[NETWORK]
max_connections = 100
message_timeout = 300

[DATABASE]
type = sqlite
path = data/neighborhood.db
```
